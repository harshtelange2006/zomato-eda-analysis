# Zomato Bengaluru Restaurant Analysis

EDA on 51,000+ restaurants listed on Zomato in Bengaluru.

## Questions Answered

- Which restaurant type is most common?
  Quick bites and Casual Dining are most common type approx 73.3% this both.
  
- Which cuisines appear most frequently?
  North Indian Cuisines appear most frequently almost 32.7%.
  
- Which locations have the most restaurants?
  BTM have most Restaurants 13.7% is highest and double digit.

- Do table booking restaurants rate higher?
  Yes.
  
- Which restaurant has the most votes?
  Byg Brewski Brewing Company	in Sarjapur Road	16832 has most votes.
  
- Does online ordering affect ratings?
  Yes. 49% sales from online order so by hypothetical 48% online order affect rating.
  
## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Dataset
Zomato Bengaluru Restaurants - Kaggle
